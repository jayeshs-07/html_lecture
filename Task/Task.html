Basic Defination

1. Website :- Collection of Web Pages you can view Using Internet & a Browser. Ex. jayeshresume.in

2. www :- World Wide Web (www) is also commonly called w3 or web. HTTP and www are used in website URL.

3. HTTP/HTTPS

  • HTTP :- Hypertext transport Protocol used for Transferring Data Over a network.

  • HTTPs :- The S HTTPS stands for secure.HTTPS Used TLS (Transport Layer security) to encrypt HTTP requests and responses.
    Ex. https://Topitcompanes

4. Clint Server :- A server is simply a computer that provides the network resources and provides service to other computers when they request it.

A client is the computer running a program that requests the service from a server. Ex. Email or www.

5. Domain :- A domain name is the address of a website. It's the name you type into a web browser to access that specific website.

the domain name for freetutorial is freetutorial.in

6. Host :- A host is a computer or other device that communicates with other hosts on a network.Also known as network hosts.

7. Hosting :- an online service that enables you to publish your website or web application on the internet.

8. FTP:- FTP is a way to download, upload, and transfer files from one location to another on the Internet and between computer systems.

9. Path :- Uniquely indentify location in a Directory structure.

10. Extension :- file name extension. Ex. ( .txt , .doc, .html) etc...

11. Editor/vscode :- a source-code editor that can be used Progaramming languages, C#, C++, java , javascrip, HTML.

12. URL :- Universal Resource locator URLs consist of multiple parts including a protocol and domain name that tell a web browser how and where to retrieve a resource.

13. Address Bar a text box in a web browser displaying the address of the web page that is currently being viewed.

14. Title Bar Top of window display application defined line, text known as Titile Bar.
